## 🔧 Troubleshooting

### Common Issues
- **Java not recognized:** Check if Java path is set correctly
- **Eclipse won't start:** Run as administrator or check Java installation
- **Programs won't run:** Ensure main method is present and file is saved

### Getting Help
- Check Eclipse error messages (red underlines)
- Verify syntax and case sensitivity
- Ensure proper curly braces placement
- Confirm semicolons at end of statements
